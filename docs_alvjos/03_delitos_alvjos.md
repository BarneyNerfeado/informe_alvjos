# Tipificacion de Delitos

- ***Articulo 2 - Acceso ilicito:*** 
Castiga al que, sin autorizacion acceda a un sistema informatico, vulnerando las medidas de seguridad.

**Mapeo de la accion:**
El atacante al utilizar las credenciales robadas de un funcionario público vulneró este artículo. Aunque la credencial era legítima, el atacante no tenía la autorización legal para utilizarla, superando las barreras de control de acceso del sistema de FICOBA.

<br>

- ***Articulo 6 - Receptacion de datos informaticos:***
Penaliza a quien comercialice, transfiera o almacene datos obtenidos de forma ilicita.

**Mapeo de la accion:**
Si el atacante descargó los 1,2 millones de registros para su posterior venta en foros o para ejecutar fraudes masivos de suplantación de identidad, el acto de conservar y lucrar con esa base de datos constituye este delito.
